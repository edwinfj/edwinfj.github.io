Project FC-NN

active

A library of fully connected neural network with arbitrary number of layers, built from scratch, using ReLU neurons and softmax cross-entropy loss with L2 norm regularization. Support batch normalization, dropout, Nesterov momentum, rmsprop, and adam.

Skills practiced:
Python, Numpy, Matplotlib

Project .klarf utilities

active

Collection of the utilities I created for manipulating klarf files, which are standard format files of results from defect inspection tools in semiconductor industry. Currently support attribute value exporting / importing.

Skills practiced:
Python, Numpy, Scipy

Project Snake War

archived

A hobbyist project I did during my undergraduate years. It is a game inspired by two classic NES games “greedy snake” and “tank war”. Graphics is based on GDI+. Tested out on Windows XP/Vista/7 and passed on all platforms. Compiled executable is provided.

Skills practiced:
C++, OOP, GDI+
